**What is AnvilXpFix?**:

In Vanilla Minecraft 1.12.2, Anvil costs are exponential, therefore as you repair or enchant an item, the cost eventually becomes "Too Expensive," and even a basic repairs/enchants  consume a massive amount of "hidden" raw experience points.

Anvil XP Fix surgically changes this logic. Instead of using exponential levels, the Anvil now calculates costs based on flat experience points like newer versions of minecraft, which is the main focus of this mod.

For some reason, I was unable to find a mod to help me with this issue for 1.12.2 specifically, so I made this mod for myself, but have uploaded it in the event it could help someone else.
